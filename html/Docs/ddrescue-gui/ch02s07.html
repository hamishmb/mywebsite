Formatting

If you format a disk, you are giving it a file system. This can also mean erasing the previous filesystem and everything it contains. If you say a disk is formatted as FAT32, it means the disk has a FAT32 filesystem on it. Often when a disk is formatted, the data on it isn't actually overwritten, and it can sometimes be recovered. The same goes for deletion.
